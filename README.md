# Cypress Automation Test Suite

E2E test projects using **Cypress** and **TypeScript**.

## Projects

| Project | Description |
|---|---|
| `project-1` | React app — navigation, forms, routing |
| `project-2` | API + UI integration tests |
| `final` | Full suite — E2E, visual, accessibility |

## Quick start

```bash
cd project-1
npm install
npx cypress open      # interactive
npx cypress run       # headless / CI
```

## CI
Tests run on every push via GitHub Actions (`.github/workflows/cypress.yml`).
